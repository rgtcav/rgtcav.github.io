Конспект лекции 2. Классификация сетей.

Компьютерные сети классифицируются по двум признакам: общие (пользовательские) и специальные (профессиональные).

Общие признаки

Выделяют следующие общие признаки:

  1. По архитектуре
  2. По масштабу администрирования
  3. По уровню однородности
  4. По способу подключения
  5. По предоставляемым сервисам(службам)
  6. По назначению
  7. По административным отношениям между компьютерами
  8. По топологии

По архитектуре

  1. LAN – Local Area Network ( ЛВС ) – Локальные вычислительные сети: Ethernet, FastEthernet, ARCnet, TokenRing , LokalTalk (Apple) , WLAN, FDDI.
  2. WAN – Wide Area Network ( РВС ) – Региональные и глобальные вычислительные сети: FDDI*, ATM, FrameRelay.
  3. MAN – Metropolian Area Network ( TВС ) – Городские (территориальные) вычислительные сети.. Это понятие в настоящее вpемя уже не используется. Оно вытеснено понятием WAN.
  4. GAN – Global Area Network ( ГВС ) – Глобальные вычислительные сети.Это понятие в настоящее вpемя уже не используется. Оно вытеснено понятием WAN.

По масштабу администрирования

  1. ЛВС рабочих групп, которые состоят из нескольких ПК, работающих под одной операционной системой. В такой ЛВС, как правило, имеется несколько выделенных серверов: файл-сервер, сервер печати.
  2. ЛВС структурных подразделений (отделов). Данные ЛВС содержат несколько десятков ПК и серверы типа: файл-сервер, сервер печати, сервер баз данных.
  3. ЛВС предприятий (фирм). Эти ЛВС могут содержать свыше 100 компьютеров и серверы типа: файл-сервер, сервер печати, сервер баз данных, почтовый сервер и другие серверы.

По уровню однородности

  1. Однородные сети, которые содержат однотипные компьютеры и системное программное обеспечение.
  2. Неоднородные сети, которые содержат разнотипные компьютеры и системное программное

По способу подключения

  1. Проводные сети - сети, в которых подключение устройств происходит с помощью кабеля.
  2. Беспроводные сети - сети, в которых подключение происходит с использованием беспроводных технологий (радиосигнал, инфрокрасное излучение, спутниковые каналы и т.п.).
  3. Комбинированные сети - сети, которые используют технологии двух предыдущих способов.

По предоставляемым сервисам

Сервис(служба) - это услуги(функции), предоставляемые пользователю сети.

Сервисов несчётное множество. Поэтому, в качестве примера можно привести такие сервисы, как получение файлов, просмотр погоды, почтовый сервис и т.п.

По назначению

  1. Вычислительные сети, предназначенные для расчетных работ.
  2. Информационно-вычислительные сети, которые предназначены, как для ведения расчетных работ, так и для предоставления информационных ресурсов.
  3. Информационно-советующие сети, которые на основе обработки данных вырабатывают информацию для поддержки принятия решений.
  4. Информационно-управляющие сети, которые предназначены для управления объектами на основе обработки информации

По административным отношениям между компьютерами

  1. ЛВС с централизованным управлением (с выделенными серверами)
  2. ЛВС без централизованного управления (децентрализованные) или одноранговые (одноуровневые) сети
